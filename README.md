⚙️ Como executar o projeto localmente

Clone o repositório.

Na pasta do backend, crie um banco PostgreSQL e configure o .env com suas credenciais. Rode npm run dev.

Na pasta do mobile, duplique o arquivo .env.example, renomeie a cópia para .env e ajuste a variável EXPO_PUBLIC_API_URL com o IP da sua máquina ou emulador.

Rode npx expo start.
